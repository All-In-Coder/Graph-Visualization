    Steps for Drawing a Graph :-

    1. Enter Number of Nodes Your Graph have

    2. Enter Number of Edges Your Graph have

    3. Choose What You want to perform

    4. Enter Source and Target Node only if you Choose Shortest Path

    4.1 If you have Choosen DFS, BFS then you can enter Starting Node

    4.2 If you have choosen Bipartite or Cycle Option, then no need to Enter Source or Target Node

    5 Enter Edges in the given textarea in (a b) form

    6 Give a name to your Graph
